To build c55-connected-audio-framework project, 

1. Make sure CCS4 is insatlled in C:\Program Files\Texas Instruments\ccsv4
2. Make sure DSP/BIOS 5.33.6 is installed in C:\CCStudio_v3.3\bios_5_33_06
3. Make sure Codegn tool 4.3.6 for CCS4 is insatlled in C:\Program Files\Texas Instruments\C5500 Code Generation Tools 4.3.6
4. Run C55_Connected_Audio_Framework-xx.xx.xx.xx-Setup.exe installer 
5. cd to the install directory and run click2build.bat
